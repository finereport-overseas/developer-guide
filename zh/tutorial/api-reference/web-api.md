# Web API

## 概述

WebAPI 是网络应用程序接口，包含了广泛的功能。

网络应用通过 API 接口，可以实现存储服务、消息服务、计算服务等能力，利用这些能力可以开发出强大功能的 Web 应用。

比如通过 WebAPI 访问到数据库里的内容，通过 GET 方式展示出来，给其他终端调用。

WebAPI 类型的接口不是后台接口，也不是 JS 接口，是 Rest 接口。

Rest 接口基于 Http 协议，可以归结为一堆 Http 请求，以 URL 形式发送请求得到后台的响应数据。
